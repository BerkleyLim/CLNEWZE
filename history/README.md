# History 파일 내용

- 내용이 길기 때문에 월별로 끊어 히스토리를 남김
- 개발 내역을 보면서 해당 날짜별로 무엇을 했는지 되집어 보는 시간을 가진다.

- 9월 이후, React 관련 CS 공부와 TypeScript, Recoil과 React-Query 사용방식 등 활용하여 적용
- Google Chart와 Naver Map 부분 공부하여 소개 부분 적용 예정
